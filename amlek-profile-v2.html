PIR Motion Detector
Detects movement via a PIR sensor on D2. Triggers an LED on D13 and streams real-time status to the Serial Monitor at 9600 baud. Foundational motion-sensing circuit.

Water Level Monitor
Uses HC-SR04 ultrasonic sensor to measure tank water level in real time. Auto-controls a relay pump — switches ON below 30% and OFF above 90%. LCD shows live percentage.
Components
Arduino UnoHC-SR04Relay ModuleLCD 16×2LED
Open on Wokwi ↗
1#include <LiquidCrystal.h> 2LiquidCrystal lcd(7, 6, 5, 4, 3, 2); 3int trig=9, echo=8, relay=10, led=11; 4long duration; int distance; 5int tankHeight = 20; 6 7void setup() { 8 pinMode(trig, OUTPUT); 9 pinMode(echo, INPUT); 10 pinMode(relay, OUTPUT); 11 pinMode(led, OUTPUT); 12 lcd.begin(16, 2); 13 lcd.print("Water Controller"); 14 delay(2000); lcd.clear(); 15} 16 17void loop() { 18 // Ultrasonic pulse 19 digitalWrite(trig, LOW); delayMicroseconds(2); 20 digitalWrite(trig, HIGH); delayMicroseconds(10); 21 digitalWrite(trig, LOW); 22 duration = pulseIn(echo, HIGH); 23 distance = duration * 0.034 / 2; 24 int level = tankHeight - distance; 25 int pct = (level * 100) / tankHeight; 26 lcd.setCursor(0,0); 27 lcd.print("Level: "); lcd.print(pct); lcd.print("% "); 28 if (pct < 30) { 29 digitalWrite(relay, HIGH); // Motor ON 30 lcd.setCursor(0,1); lcd.print("Motor: ON "); 31 } else if (pct > 90) { 32 digitalWrite(relay, LOW); // Motor OFF 33 lcd.setCursor(0,1); lcd.print("Motor: OFF "); 34 } 35 delay(1000); 36}

LDR Auto-Brightness Control
Reads ambient light via an LDR (photoresistor) on A0. Inversely maps the 0–1023 ADC reading to a PWM brightness (0–255) on pin D9. Bright environment dims the LED; darkness brightens it automatically.
Components
Arduino UnoLDR / PhotoresistorLED10kΩ Resistor220Ω Resistor
Open on Wokwi ↗
1// LDR Auto-Brightness Control 2// LDR A0 → A0 | LED PWM → D9 3 4const int ldrPin = A0; // Photoresistor 5const int ledPin = 9; // PWM LED pin 6 7void setup() { 8 pinMode(ledPin, OUTPUT); 9 Serial.begin(9600); 10} 11 12void loop() { 13 int lightValue = analogRead(ldrPin); 14 15 // Invert: bright → dim LED, dark → bright LED 16 int brightness = map(lightValue, 0, 1023, 255, 0); 17 brightness = constrain(brightness, 0, 255); 18 19 analogWrite(ledPin, brightness); 20 21 Serial.print("Light Sensor Value: "); 22 Serial.print(lightValue); 23 Serial.print(" | LED Brightness: "); 24 Serial.println(brightness); 25 26 delay(200); 27}
